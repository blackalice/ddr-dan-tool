# RemyWiki songlist snapshots

Saved RemyWiki arcade songlist pages live here as `.html` or `.htm` files for
reference. Songlist override data is maintained as static JSON in
`data/ddr-ver/` and copied to `public/ddr-ver/` by `npm run sync:public`.

If a future DDR release changes these lists, use the saved pages as source
material for a one-off update and commit the resulting JSON files rather than
adding RemyWiki parsing to the normal data pipeline.
