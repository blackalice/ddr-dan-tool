DanceDanceRevolution 4thMIX

Release Information

  • Release Date: August 24th, 2000.
  • Soundtrack release date:
    • DanceDanceRevolution 4thMIX ORIGINAL SOUNDTRACK: March 16th, 2001.
  • Slogan: 見せよ！ 魂揺さぶるマクシマム・ムービン！！ (Miseyo! Tamashii yusaburu MAXIMUM MOVIN'!!, Show me! The soul shaking maximum movin'!!)

General Information / Changes

  • First mainline DanceDanceRevolution game to run the menu interface in 480i mode. Gameplay still runs in 240p mode.
  • New interface replaces the jukebox style from the first three games.
  • First DanceDanceRevolution game where selection of play styles are available in the player selection menu, instead of using codes to reveal secret menu options on the Caution Screen.
    • The song list is divided in categories, and each category is represented by two characters.
    • On the song selection screen, the songs' banners appear tilted on the songwheel and their backgrounds are visible over it. Songs can be navigated from left to right.
    • First DanceDanceRevolution game to utilize color labeling to denote the origin of a particular song.
      • Green denotes a song that is brand new for 4thMIX or debuted in 3rdMIX PLUS.
      • Yellow denotes an unlockable song.
      • Blue denotes all other older songs.
  • Due to the new scoring system, the silver S, golden S, SS and SSS ratings from DanceDanceRevolution 3rdMIX PLUS have been removed, replaced by sparkling A when achieving full combo and AA when achieving perfect full combo.
    • The summary of maximum number of played songs and allocated scores in every after gameplay was removed until DanceDanceRevolution 5thMIX.
  • First DanceDanceRevolution game to enlist available song titles on the ending credits, replacing the "THANK YOU FOR PLAYING" screen.
  • First non-CLUB VERSION DanceDanceRevolution game to include BEMANI crossovers, instead of remixes.
  • First DanceDanceRevolution game with hidden songs that can be unlocked by the arcade operator.
  • First mainline DanceDanceRevolution game to have 6-Panel charts, introduced in the Solo series.
  • First DanceDanceRevolution to introduce BATTLE charts. Both players use separate target arrows, but the arrows scroll starting from the bottom center and then split off, either to both sides or to only one side.
    • A different version of BATTLE mode appears in DanceDanceRevolution SuperNOVA.
  • First DanceDanceRevolution game to have five songs courses instead of four, although this must be enabled in the operator's menu first.
  • Every song from DanceDanceRevolution and DanceDanceRevolution 2ndMIX received Double MANIAC charts.
  • KONAMI Edit Data charts for Solo mode were added, based on their respective Double charts.
  • ANOTHER difficulty has been renamed to TRICK. This name was first used in the non-Japanese releases of DanceDanceRevolution 3rdMIX.
  • A difficulty menu has been added that allows the selection of the three primary difficulties, as well as additional charts such as the BATTLE charts mentioned above and installed edit data. This menu is displayed after selecting a song. This practice was used in many European Dancing Stage titles ranging from Dancing Stage EuroMIX to Dancing Stage Fusion, and would be later used in the main series again starting with DanceDanceRevolution (2014).
  • The announcer's voice has been lowered in pitch and is synthesized to sound more echo like.
  • First DanceDanceRevolution game to use iterative playback samples of a selected song. 3rdMIX and earlier use one time playback samples of a selected song.
  • In the Asian release of DanceDanceRevolution 4thMIX, LOVE, GET UP(BEFORE THE NIGHT IS OVER), GET UP AND DANCE, and the Korean songs from DanceDanceRevolution 3rdMIX VER.KOREA or 3rdMIX PLUS were not included.
    • Missing songs from the Asian version of DanceDanceRevolution 3rdMIX are marked as "NEW!", which are: Stomp to my beat, TUBTHUMPING, IF YOU CAN SAY GOODBYE, and XANADU.
    • All hidden songs are missing as well, due to this version having no code system.
  • New songs: 49 (including 7 from the Solo series)
  • Total songs: 136 (Japan), 126 (Asia)

Unlock Information

Note: These codes are only for the Japanese version. The Asia version has no code system for unlocking the 12 hidden songs. (However, if the machine has a Memory Card slot, these can be played via LINK MODE NONSTOP "ORDER" courses if the hidden songs were set in NONSTOP ORDER from the PlayStation Arcade Link System. If one of the songs set was not present in the Asian version of the game, it will be changed to ONLY YOU.)

To unlock the hidden songs in DanceDanceRevolution 4thMIX, the arcade operator needs to access Test Mode.

  1. Press Test button to enter the Service Menu.
  2. Highlight "Game Options".
  3. Hold 1P down panel and service button, and then press 1P start button. The "secret" menu will be revealed.
  4. Select "Append".
  5. Input the password in order as shown below. Use 1P left and right selection buttons to select spaces, and 1P start button to change characters.
  6. After entering the password, hold 1P left and right selection buttons, and then press 1P start button to confirm the password.
  7. Select "Exit" to game options menu. Do not select "Initialize".
  • To unlock 1st PASS only: R6HGS4T-81XEB5I.
  • To unlock 1st PASS and 2nd PASS: M67BKPT-YLPNS8N
  • To unlock 1st PASS, 2nd PASS and 3rd PASS (FINAL): OGLAPLT-JC2LO8I
